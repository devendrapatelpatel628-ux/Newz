# TeleManager Python-only Build

This build removes React/Vite/Node/npm from the app. The full website is generated by Python with FastAPI HTML responses.

## Included

- Python FastAPI backend API
- Python-generated website frontend
- Account login + OTP + 2FA pages
- Dashboard
- Proxy manager
- Report manager
- Analytics
- Account pool
- Evidence collector
- Warmup page
- Scheduler status page
- Railway-ready Dockerfile

## Not Included

- React
- Vite
- TypeScript
- Node.js frontend
- npm build step
- JavaScript frontend files

## Local Run

```bash
pip install -r requirements.txt
python main.py
```

Open:

```text
http://localhost:8000
```

API health:

```text
http://localhost:8000/api/health
```

## Railway Deploy

1. Push this folder to GitHub.
2. Create a Railway project.
3. Deploy from that GitHub repo.
4. Railway will use the root `Dockerfile`.
5. Open the Railway public URL.

Railway automatically provides the `PORT` variable. The existing `config.py` reads it.

## Persistence Note

Telegram sessions and SQLite data are stored in:

```text
/app/sessions
/app/data
```

For long-term sessions on Railway, attach a Railway volume if available.
